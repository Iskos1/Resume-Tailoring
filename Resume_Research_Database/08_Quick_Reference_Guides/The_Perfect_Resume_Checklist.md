### **Overall Structure**
- [ ] Section titled "Experience," "Professional Experience," or "Work Experience"
- [ ] Reverse chronological order (most recent first)
- [ ] All positions from last 10-15 years included
- [ ] Older positions consolidated or omitted if not relevant
- [ ] No unexplained gaps >6 months

### **Each Job Entry Includes**
- [ ] Job title clearly visible
- [ ] Company name included
- [ ] Location (City, State)
- [ ] Dates (Month Year â€“ Month Year format)

### **Job Entry Formatting**
- [ ] Job title in BOLD

- [ ] Company name on second line (or same line as title)
- [ ] Consistent format for all jobs
- [ ] Clean, scannable layout

### **Bullet Points - Quantity**
- [ ] Current role: 4-6 bullets
- [ ] Previous role: 3-4 bullets
- [ ] Older roles: 2-3 bullets
- [ ] No job has >7 bullets (cognitive overload)
- [ ] No job has <2 bullets (unless very brief role)

### **Bullet Points - Quality**
- [ ] Every bullet starts with action verb (not "Responsible for")
- [ ] Past tense for previous roles
- [ ] Present tense for current role ONLY
- [ ] No personal pronouns (I, me, my)
- [ ] Consistent bullet symbol throughout (â€¢ or â€“)
- [ ] Each bullet is 1-2 lines maximum

### **Bullet Points - Quantification (CRITICAL)**
- [ ] 80%+ of bullets include numbers/metrics
- [ ] Quantification includes:
  - [ ] Revenue/cost impact ($)
  - [ ] Percentages (growth, improvement)
  - [ ] Team/scope size
  - [ ] Timeframes
  - [ ] Volume/scale
- [ ] Before/after comparisons when possible
- [ ] Specific numbers (not "many," "several," "significant")

### **Bullet Points - Structure**
- [ ] Follows CAR/STAR/XYZ formula:
  - [ ] Context (brief setup)
  - [ ] Action (what you did)
  - [ ] Result (quantified outcome)
- [ ] Result-first format when possible (leads with impact)
- [ ] Each bullet demonstrates value, not just duties
- [ ] Technical skills/tools mentioned naturally

### **Achievement Hierarchy**
- [ ] First bullet per job = biggest achievement
- [ ] Bullets ordered by impact (most impressive first)
- [ ] Demonstrates progression across roles
- [ ] Shows increasing responsibility over time
