## How to Use

1. Visit the live site
2. Enter your OpenAI API key (get one at https://platform.openai.com/api-keys)
3. Paste your master resume
4. Paste a job description
5. Click "Tailor My Resume"

Your API key is stored locally in your browser and is only sent to OpenAI's servers.

## Local Development

Simply open `index.html` in a web browser. No build process required!
